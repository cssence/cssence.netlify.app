# cssence.netlify.app

This site is primarily a collection of web pages for testing purposes.

See live version under https://cssence.netlify.app/.

## Setup

To be used as is, zero dependencies, no build script.

The `/public` folder is source and hosted version at the same time. Ideally, point an [http-server](https://www.npmjs.com/package/http-server) of your choice there.
